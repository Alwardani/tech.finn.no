## Usage

We are using Jekyll. Read more about it [here](http://jekyllrb.com/).

```sh
$ gem install bundler # only needed once
$ bundle install
$ bundle exec jekyll serve -w
```

Open <http://localhost:4000> in your browser, and voil√†.
